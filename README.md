# Usage

## Local Build & Launch

Install [docker-compose](https://docs.docker.com/compose/install/)
'''git clone https://github.com/cyprigroup/jrapitest.git'''
'''cd jrapitest'''
'''docker-compose up'''
- Open a browser to: [http://127.0.0.1:8888/lab?token=JR](http://127.0.0.1:8888/lab?token=JR)
